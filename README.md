## 介绍

Plunge是通过识别图表的形态量价来选股的工具,主要用来参考学习,不作为投资依据

## 准备

- [Python](https://www.python.org/) - 行情数据入库使用python开发
- [Golang](https://go.dev/) - 识别图表使用Go语言开发
- [MariaDB](https://mariadb.org/) - 数据库使用MariaDB
